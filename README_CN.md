## 📖 历史与致敬

**"代码会过时，但记忆永存。"**

本项目的灵感来源于一款名为 **EpicMobs 1.3 Bate**（是的，原文件名就是 Bate 而不是 Beta）的古老插件，它大约完成于 **2017年8月11日**。

* **原作者:** 南柯一梦 (腐竹 瑾辰)
* **原修改者:** 凯文 (QQ: 2846339392)
* **原服务器:** `mc.toin.cc`

我和这个插件的故事始于 **2019年8月27日**，那是 **ChinaMC**（中国版图粘液科技服务器）的黄金时代。随后在 **2023年** 的 **Lom 联盟三周目** (1.16.5 粘液科技) 中，这份记忆再次被唤醒。

时间来到 **2025年**，当 **cnwanchen** 复活了这个服务器时，我意识到虽然原插件已在近十年前停止更新，但它带给我们的乐趣不应就此消失。凭借我对计算机科学的深入理解，我决定解构并从零重写这个插件。

**EpicMobsReborn** 是对 **ChinaMC**、**Lom联盟** 以及所有陪伴我们度过青春的服务器的一份献礼。

## ✨ 插件特性

本插件致力于为服务器带来 RPG 般的挑战体验：

* **距离决定难度**: 怪物离出生点越远，血量越厚。
    * *从此告别千篇一律的刷怪体验，远方的荒野真正充满了危险。*
* **动态称号系统**: 怪物头顶显示 `[精英]`、`[史诗]`、`[真圣]` 等霸气称号。
* **装备增强**: 高等级怪物会自动装备附魔武器（钻石剑、力量弓等），给玩家带来实质性威胁。
* **精英技能**: 20级以上的精英怪在受到攻击时，有概率**瞬移**突袭玩家，增加战斗博弈感。
* **显血提示**: 通过 Action Bar 实时显示怪物剩余血量，战斗反馈更直观。
* **全面适配**: 完美支持 1.16.5 的新生物（猪灵蛮兵、劫掠兽、幻翼等）。
* **高度可配置**: 支持多语言切换（简中/英文），支持自定义难度系数。

## 🛠️ 安装与兼容性

### 环境要求
* **Java 版本:** Java 16 或更高（这是 1.16.5 Paper 端的标准要求）。
* **服务器核心:** 强烈推荐使用 **Paper 1.16.5**。
    * *兼容:* Spigot, Purpur 等基于 Bukkit 的服务端。
    * *不兼容:* Fabric, Forge (除非使用 Mohist/Cardboard 等混合端，但不保证稳定性)。

### 安装步骤
1.  下载 `.jar` 文件。
2.  将其放入服务器的 `plugins` 文件夹。
3.  重启服务器。
4.  (可选) 修改 `config.yml` 中的 `Language: zh_CN` 确保语言正确。

## ⚙️ 命令与配置

重载配置文件与语言包：
```bash
/epicmobs reload
```

*(需要权限: epicmobs.reload)*

你可以在 `config.yml` 中调整怪物的**血量成长系数**，或者关闭特定世界的增强功能。
